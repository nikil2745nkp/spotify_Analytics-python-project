# 🎵 Spotify Data Analytics Dashboard

An interactive Data Analytics project built using **Python** and **Streamlit** that provides deep insights into Spotify music data through dynamic visualizations and smart exploration tools.

---

## 🚀 Features

- 📊 **Dataset Overview**
  - View dataset structure, columns, and statistics
  - Filter and download selected data

- 📈 **Interactive Dashboard**
  - KPI metrics (Songs, Artists, Popularity, Streams)
  - Genre-wise performance analysis
  - Language & label distribution

- 🎶 **Music Analytics**
  - Top artists and songs
  - Popularity vs Duration analysis
  - Time trend analysis (songs over years)
  - Hierarchical visualizations (Sunburst, Treemap)

- 🤖 **Data Assistant**
  - Ask questions about dataset
  - Get instant insights with charts

- 🔍 **Smart Search Engine**
  - Search songs directly on Spotify

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit
- Pandas
- Plotly
